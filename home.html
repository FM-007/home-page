Sobre Mim !!!

Atualmente atuando como desenvolvedor Java com Spring Framework na construção de micro serviços voltados ao backend do setor financeiro e ambiente Cloud em projetos ágeis. Possuo sólida experiência como desenvolvedor frontend em React.js, consumindo APIs em backend Java. Possuo perfil automotivado e cooperativo, reconhecido pela flexibilidade e criatividade na solução de problemas.


Participei da construção de micro serviços nos projetos Next e Bradesco utilizando Java 8 e 11, Spring Boot, SQL Server voltado a Cloud Azure. Atualmente atuando em um projeto que integra o chat de um banco digital ao sistema de Inteligência Artificial, através de um micro serviço que recebe informações da plataforma de atendimento via Webhook, filtramos as requisições que devem ser direcionadas para a IA e retornamos a resposta através de uma API. Atuei também na construção de interfaces gráficas utilizando React com TypeScript e na manutenção e migração de interfaces construídas em Angular para o React.


Contribui no desenvolvimento de aplicações web, voltadas a instituições financeiras, utilizando React, JavaScript, Sass(scss), Axios, Redux, React Hook's, Context API, consumindo API's Rest construídas utilizando Java 11, Spring Boot, SQL Server voltado a Cloud AWS.
